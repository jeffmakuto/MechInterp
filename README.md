
# 🚀 Mechanistic Interpretability Research

## 🎯 Objective

Build deep, foundational expertise in **Mechanistic Interpretability of LLMs**, enabling original research contributions, open-source tooling, and impactful publications in interpretability, AI safety, and transparency.

## 🔬 Mechanistic Interpretability Toolkit

| Technique | Description |
|----------|-------------|
| **Activation Patching** | Replace activations from corrupted run with clean run to localize behavior |
| **Logit Lens** | Project residual stream onto final logits to interpret intermediate computations |
| **Direct Logit Attribution (DLA)** | Decompose the final logits into linear contributions from each layer/component |
| **Sparse Autoencoders** | Extract disentangled latent features from neuron activations |
| **Path Patching / Ablation** | Trace flow of information through network or break components to test causality |
| **Visualization** | Attention heatmaps, neuron activation graphs, feature attribution plots |
